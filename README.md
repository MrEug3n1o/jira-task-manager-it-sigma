# Sigma Task Manager

**Sigma Task Manager** is a powerful web application designed to help teams collaborate efficiently on multiple projects. It allows users to **create, assign, and manage tasks** with ease while maintaining a **clean and modern interface** for an enjoyable workflow experience.  

With Sigma Task Manager, you can:
- Organize projects and track progress in real time  
- Collaborate with teammates seamlessly  
- Enjoy an intuitive, visually appealing dashboard
